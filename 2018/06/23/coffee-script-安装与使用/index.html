coffee script 安装与使用

发表于 2018-06-23 | 分类于 前端开发 |

CoffeeScript 是一门编译到 JavaScript 的小巧语言. 在 Java 般笨拙的外表下, JavaScript 其实有着一颗华丽的心脏. CoffeeScript 尝试用简洁的方式展示 JavaScript 优秀的部分.

安装

CoffeeScript 编译器本身是 CoffeeScript 写的, 使用了 Jison parser generator。 命令行版本的 coffee 是一个实用的 Node.js 工具. 不过编译器并不依赖 Node, 而是能运行于任何 JavaScript 执行环境。

安装前你需要最新稳定版 Node.js, 和 npm 或者 yarn. 借助 npm 可以安装 CoffeeScript，需要注意的是 coffeescript 而不是 coffee-script

通过 coffee -v 选项来输出版本号，不出意外，应该是安装成功的。

命令行参数

使用 coffee -h 来查看命令行选项。

coffee 提供了一个交互式命令行，可以快速验证一些代码片段。
