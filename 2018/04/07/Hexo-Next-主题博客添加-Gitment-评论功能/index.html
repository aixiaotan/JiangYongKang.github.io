Hexo Next 主题博客添加 Gitment 评论功能

发表于 2018-04-07 | 分类于 环境搭建 |

Gitment 是作者实现的一款基于 GitHub Issues 的评论系统。支持在前端直接引入，不需要任何后端代码。可以在页面进行登录、查看、评论、点赞等操作，同时有完整的 Markdown / GFM 和代码高亮支持。尤为适合各种基于 GitHub Pages 的静态博客或项目页面。

现在最新版本的 Next 主题已经不需要自己手动去繁琐的添加代码样式了。只需要在 Next 的配置文件中简单配置一下就行了。

注册 OAuth Application

在你自己的 GitHub 中注册一个 OAuth Application 填写内容任意，但是 callback url 要注意一点，一般都是填写自己的 GitHub 地址，比如我的就是 https://github.com/JiangYongKang。
填写完之后，会生成一个 client id 和 client secret。

页面发布后，你需要访问页面并使用你的 GitHub 账号登录，点击初始化按钮。Gitment 会自动的帮你在仓库中创建 issues 记录文章的评论。
