Ruby 代码块的使用

发表于 2017-10-05 | 分类于 后端开发

定义和使用块

通过 do end 关键字声明一个代码块，通过 & call 符号调用块。块是不可以单独使用的，必须配合方法调用。

块参数

方法可以向块传递一个或多个参数，多个入参，必须用 | ... | 包裹。最常用的情况是，一个方法对应一个代码块。如果想实现定义一个块在多个方法中调用，或者一个方法调用多个块。可以通过 Ruby procs 来实现。

使用 yield 关键字

yield 关键字可以调用随方法调用提供的块，方法名必须和块的名字一致。

块的格式

do ... end 和 { ... } 都可以声明代码块，通常情况下，能够一行写完的代码用 { ... } 包裹。多行代码用 do ... end 包裹。

each 方法

最常见的 each 循环使用的就是块。
