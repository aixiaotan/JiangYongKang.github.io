Ruby on Rails 静态资源文件的加载

发表于 2017-09-08 | 分类于 后端开发

加载顺序

  • 默认情况下加载 application.js。
  • 根据请求的 Controller，加载对应的 JS 文件。例如：请求 localhost:3000/users，会加载 app/javascripts/user.js
  • 每一个 JS 文件可以加载其他的 JS 文件。

在 ERB 模板中引入 JS

如果需要在 erb 模板文件中通过 <%= javascript_include_tag "main" %> 标签单独加载某个 js 文件。

在 JS 文件中加载其他的 JS 文件

  • 每一个 JS 文件也都可以加载其他的 JS 文件。
  • 在 JS 文件中 //= 开头的代码行对于 Rails 来说并不是注释，它会解析这一行代码。引入其他的 JS 文件。
  • JavaScript 解释器仍然将 //= 开头的代码当做注释来处理。
