安装 ruby 版本管理工具 rbenv

Mac 是自带了 ruby 的，但是对于版本的管理不是那么灵活，所以我们这里用 rbenv 来对 ruby 的版本进行管理。虽然使用 rvm 也差不多，不过我个人更加喜欢 rbenv 一些。推荐使用 brew 安装。

如果一切正常输入 rbenv -v 应该能看到版本号。
