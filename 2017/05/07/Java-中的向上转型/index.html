Java 中的向上转型

发表于 2017-05-07 | 分类于 后端开发 |

在 Java 继承体系中，认为基类在上层，导出类在下层，因此向上转型的意思就是把子类对象转成父类类型，即将父类的引用指向子类对象。向下转型的意思就是把父类对象转成子类类型，即将子类的引用指向父类对象。
