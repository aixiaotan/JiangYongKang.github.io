Ruby 中的 AOP

发表于 2017-12-19 | 分类于 后端开发 |

Ruby 没有类似于 Java 中的 AspectJ 这样的东西，但是利用 include 和 prepend 或者是 before_action、after_action 同样可以达到 AOP 的效果。就以最为典型的，在 方法调用的前后打印日志，来简单实现一下。

before_action、after_action

rails 中的 before_action、after_action、around_action 正好和 Spring AOP 的 @Before @After @Around 很类似，唯一的不同是，在 Rails 中这些过滤器都是写在 controller 中的，而不是像 Java那样通过定义切入点去通知。也就意味着，假如我想打印所有的请求的参数，就必须在每一个 controller 中去定义，并不是像 Java 那样，通过定义一个 controller 层的切入点，就可以实现全部 controller 都被监管起来。

include、prepend

利用 include prepend 也可以做到 AOP 的效果。
